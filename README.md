# Adaptive Causal Digital Twin  
**A full-stack, cloud-ready platform for building adaptive, causal-aware digital twins with simulation engines, causal discovery, RL/BO policy optimization, real-time data ingestion, and 3D geospatial visualization.**

![banner](docs/assets/banner.png)

---

## ‚≠ê Overview

The **Adaptive Causal Digital Twin** is an end-to-end platform for modeling, simulating, understanding, and optimizing complex systems such as:

- cities  
- supply chains  
- networks / infrastructure  
- manufacturing systems  
- healthcare systems  
- energy grids  
- mobility / transportation  

Unlike traditional digital twins that only *predict*, this twin can also **understand why**, via causal reasoning, and **recommend optimal interventions**, via reinforcement learning and Bayesian optimization.

## üöÄ Key Capabilities

### **1. High-Fidelity Simulation Engine**
- Agent-based modeling (ABM)  
- Discrete-event simulation (SimPy)  
- Hybrid multi-layer simulation  
- Parallel simulation support  

### **2. Causal Intelligence Layer**
- Causal discovery (NOTEARS, FCI, PCMCI, DAG bootstrap)
- Causal inference (DoWhy, EconML)
- Automated structural causal model (SCM) builder

### **3. Surrogate Modeling**
- Gaussian processes  
- Neural surrogate models  
- Fast real-time simulation via model compression

### **4. Policy Optimization**
- Reinforcement Learning (Stable-Baselines3)
- Bayesian Optimization (BoTorch / Ax)
- Multi-objective decision optimization

### **5. Real-Time Data Ingestion**
- Kafka streaming  
- REST ingestion layer  
- Timeseries ETL and validation  

### **6. Full Front-End Digital Twin Dashboard**
- React + MapboxGL  
- 3D visualization (Three.js + Deck.gl)  
- Real-time heatmaps, flows, agents  
- Scenario & intervention editor  

### **7. Production Deployment**
- Docker  
- Docker Compose  
- Kubernetes (Helm chart included)  
- GitHub Actions CI/CD
